PizzaByte (Global Game Jam 2018)



Download Link: PizzaByte

Description:

In a world where food is no longer delivered but transmitted through code you are the owner of a new pizza shop, and it is time to create nodes and transmit your code to your hungry customers. How you structure your node towers and your coding skills will determine whether you encounter financial success or go bankrupt, losing everything you own.


Instructions:

Click the download link, download the folder, extract the contents, and run the file GameJam2018_VGDC.exe. Create topping towers used to transmit your pizza code through the city. Choose an order (represented by an exclamation mark). Type in the code to transmit the pizza before the order expires. The distance the code has to travel and owning the proper topping towers determines the length of the code.
